3. Hosting über GitHub

Diese Website wird über GitHub Pages gehostet. Beim Besuch der Website können durch den Hosting-Anbieter technische Informationen verarbeitet werden, beispielsweise:

  • IP-Adresse

Weitere Informationen finden Sie in der Datenschutzerklärung von GitHub.

4. Kontaktformular

Wenn Sie uns über das Kontaktformular eine Nachricht senden, werden die von Ihnen eingegebenen Daten verarbeitet, um Ihre Anfrage zu bearbeiten.

Dabei können folgende Daten verarbeitet werden:

  • Name
  • E-Mail-Adresse
  • Inhalt der Nachricht

Die übermittelten Daten werden ausschließlich zur Bearbeitung Ihrer Anfrage verwendet und nicht ohne Ihre Zustimmung weitergegeben.

Die Daten werden spätestens nach 90 Tagen gelöscht, sofern keine gesetzliche Aufbewahrungspflicht besteht oder eine weitere Zusammenarbeit erforderlich ist.

5. Kommunikation per E-Mail

Wenn Sie uns per E-Mail kontaktieren, werden Ihre Angaben zur Bearbeitung Ihrer Anfrage gespeichert.

Die Speicherung erfolgt ebenfalls für maximal 90 Tage, sofern keine weitere Zusammenarbeit entsteht oder gesetzliche Pflichten entgegenstehen.

6. WhatsApp-Kontakt

Auf dieser Website besteht die Möglichkeit, über einen WhatsApp-Link Kontakt mit uns aufzunehmen.

Beim Anklicken des WhatsApp-Buttons werden Sie zu WhatsApp weitergeleitet. Dabei können personenbezogene Daten, zum Beispiel Ihre IP-Adresse, an WhatsApp bzw. Meta übertragen werden.

Die Nutzung von WhatsApp erfolgt freiwillig und auf eigene Verantwortung.

Weitere Informationen finden Sie in der Datenschutzerklärung von WhatsApp.

7. Webanalyse mit GoatCounter

Diese Website verwendet GoatCounter, einen datenschutzfreundlichen Webanalysedienst zur statistischen Auswertung der Besucherzahlen.

GoatCounter arbeitet ohne Cookies und dient ausschließlich dazu, die Nutzung der Website anonym auszuwerten, beispielsweise:

  • Anzahl der Seitenaufrufe
  • Herkunft der Besucher
  • verwendete Geräte und Browser
  • beliebte Inhalte der Website

Dabei werden keine personenbezogenen Profile erstellt.

Weitere Informationen finden Sie in der Datenschutzerklärung von GoatCounter.
