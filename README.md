# AgRup Media Website

Statische Website fuer AgRup Media.

## Inhalt

- `index.html` - komplette Website
- `assets/` - Logos, Bilder und Hintergrundgrafiken

## Lokal ansehen

Die Datei `index.html` kann direkt im Browser geoeffnet werden.

## Bei GitHub speichern

1. Auf GitHub ein neues Repository erstellen, zum Beispiel `agrup-media-website`.
2. Diese Dateien hochladen:
   - `index.html`
   - `assets/`
   - `README.md`
3. Aenderungen spaeter am besten ueber Branches und Pull Requests bearbeiten.

## Hosting ueber GitHub Pages

1. Im Repository auf `Settings` gehen.
2. Links `Pages` auswaehlen.
3. Bei `Build and deployment` einstellen:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
4. Speichern.

GitHub erzeugt danach eine oeffentliche Website-Adresse. Fuer eure eigene Domain `agrupmedia.de` wird die Domain ebenfalls unter `Settings > Pages` eingetragen. Die konkreten DNS-Werte zeigt GitHub dort an und diese werden dann beim Domain-Anbieter hinterlegt.

## Kontaktformular

Die Website ist statisch. Fuer ein echtes Formular braucht sie einen Formular-Dienst wie Formspree oder spaeter ein kleines Backend. Die Zieladresse fuer Anfragen ist:

`info@agrupmedia.de`
